ajaxupload

通过AJAX来实现文件异步上传

ajax文件上传
参数:
名称 类型 属性 描述
arg1 String <任意>
 上传URL
arg2 String <任意>
 fileinput 的ID
arg3 Callback <任意>
 上传成功的回调函数
arg4 Callback <任意>
 上传失败的回调函数
版本:
  • 4.2.5
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
var Tool=require("$PATH/lib/base/util");
Tool.uploadfile(URL, ID,function(data) {
		alert(data);
	},function(err){
		alert(err);
	});

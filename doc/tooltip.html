tooltip

概述

受到Jason Frame开发的jQuery.tipsy插件的启发，我们才把这个工具提示插件做的更好，而且此插件不依赖图片，只是使用CSS3来实现动画效果，并使用data属性存储标题。

作用
工具提示插件
参数:
名称 类型 属性 默认 描述
animation boolen <任意>
 true 应用一个CSS过渡到工具提示
html boolen <任意>
 false 插入HTML工具提示。
placement String <任意>
 top 如何定位工具提示
selector String <任意>
 false 如果提供了一个选择器,工具提示对象将委派给指定的目标。
title String <任意>
 null 如果标题属性不存在默认标题值
trigger String <任意>
 hover 工具提示是如何触发
delay Number <任意>
 0 延迟显示和隐藏工具提示(ms)
并不适用于手动触发类型
如果提供,延迟应用隐藏/显示
container boolen <任意>
 false 标题内容
版本:
  • 2.0.1
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
require("$PATH/lib/util/tool/tip");
$('#example').tooltip(options);

validate

插件的核心思想就是把所有的验证条件及验证提示信息绑定到每个表单元素，让验证代码在执行时只是核对表单下各元素的值是否跟绑定的验证条件相符，这样你可以随便添加或者去掉任一表单元素而不必修改验证代码，从而使仅用一行代码去完成整站的表单验证的梦想成为现实.

表单验证

//js示例:
// $PATH为索引路径

//在HTML中的属性
  //@max 字符最大长度 默认：9999
  //@min 字符自小长度 默认：0
  //@reg 过滤用的正则表达式

  //@data-placement 显示位置 默认top
  //注意 input标签要包含在div中,标签中data-参数，参照tooltip API

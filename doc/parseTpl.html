parseTpl

解析模版tpl。当data未传入时返回编译结果函数； 当某个template需要多次解析时，建议保存编译结果函数， 然后调用此函数来得到结果

对象模板渲染
参数:
名称 类型 属性 描述
arg1 String <任意>
 模板字符串
arg2 Object <任意>
 数据源
版本:
  • 4.2.5
作者:
  • susu(744276721@qq.com)
返回值:
渲染好的模板
类型:
String

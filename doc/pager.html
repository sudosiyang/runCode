pager

分页插件，支持异步请求

分页插件
参数:
名称 类型 属性 默认 描述
parent String <任意>
 放置插件的容器
current Number <任意>
 1 设置当前页
item Number <任意>
 10 每页显示数量
total Number <任意>
 数据总数
display_num Number <任意>
 7 按钮最多显示数量
selectChange Callback <任意>
 选项更改时触发的事件
版本:
  • 2.2.5
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
var Pager=require("$PATH/lib/util/pager/page");
Pager.init({
		parent: ".pager",
		current: 1,
		item: 10,
		total: 270,
		display_num:5,
		selectChange: function() {
			console.log(this.current);
		}
	});

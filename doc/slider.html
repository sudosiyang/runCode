slider

概述

可设置滑动值和水平/垂直方向的Slider控件

作用
滑块控制封装文件
参数:
名称 类型 属性 默认 描述
parent Number <任意>
 插件生成容器（节点）
min Number <任意>
 0 刻度最小值
max Number <任意>
 100 刻度最大值
onchange Callback <任意>
 滑动值更改时回调函数
版本:
  • 1.0.3
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
var Slider=require("$PATH/lib/util/slider/slider");
Slider.init({
	parent: ".slider",
	min: 20,
	max: 200,
	onChange: function() {
		console.log(this.value);
	}
  });
// 设置滑块位置
var Slider=require("$PATH/lib/util/slider/slider");
Slider.setValue(75);

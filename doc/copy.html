copy

概述

复制插件，需要在服务器环境下使用

作用
复制插件
参数:
名称 类型 属性 描述
arg1 String <任意>
 复制按钮
arg2 String <任意>
 复制对象
arg3 Callback <任意>
 成功复制的回调函数
版本:
  • 4.2.5
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
var Tool=require("$PATH/lib/base/util");
Tool.copy(".copy", ".word", function() {
		alert("复制成功！");
	});
// HTML
<div class='word'>这是要被复制的内容</div>
  <a href='#' class='copy'>复制</a>

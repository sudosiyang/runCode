easyPieChart

概述

此插件主要是用来统计新的访问、跳出率、服务器负载、使用的RAM等，功能很强大，带有HTML5的动画效果，效果非常炫，IE9以下需要调用excanvas.js

作用
仪表状态图
参数:
名称 类型 属性 默认 描述
barColor String <任意>
 #ef1e25 该curcular栏的颜色。您可以通过有效的CSS颜色字符串，
或一个函数， 它目前的百分比值，
并返回一个有效的CSS颜色字符串。
trackColor String <任意>
 #f2f2f2 轨道的颜色，或false来禁用渲染
scaleColor String <任意>
 #dfe0e0 刻度线的颜色，false来禁用渲染。
scaleLength Number <任意>
 5 刻度线的长度（降低了图表的半径）
lineCap String <任意>
 round 定义其样式
参数为: butt , round and square
size Number <任意>
 尺寸
rotate Number <任意>
 0 旋转图标的角度
lineWidth Number <任意>
 3 轨道宽度
onStep Callback <任意>
 在动画提供的当前值(from, to, currentValue)
版本:
  • 2.0.1
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
require("$PATH/lib/util/pieChart/chart");
 $(".chart").easyPieChart({
		easing: "easeOutBounce",
		onStep: function(from, to, percent) {
			$(this.el).find(".percent").text(Math.round(percent));
		},
		lineCap: "square",
		size: 150,
		lineWidth: 20,
		barColor: "#e66e1e",
		scaleColor: false
	});

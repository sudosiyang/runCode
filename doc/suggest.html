suggest

搜索框，登陆框都时常需要进行提示输入来获取更快的查询方法

建议提示插件
参数:
名称 类型 属性 描述
target String <任意>
 绑定的输入框
suggest Array <任意>
 建议数组
版本:
  • 2.2.0
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
var Suggest=require("$PATH/lib/util/suggest/suggester");
Suggest.init({
		target: ".sug",
		suggest: ["", "@163.com", "@gmail.com", "@qq.com", "@111.com", "@tetequ.com"]
	});
// HTML
<input class='sug'/>

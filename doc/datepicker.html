datepicker

概述

快速解决时间选择问题,轻量级

作用
日历插件
参数:
名称 类型 属性 默认 描述
eventName String <任意>
 click 所需的事件来触发日期选择器
date String/Date/Array <任意>
 选定的日期（S）为字符串和Date对象的单一选择，
字符串或日期对象数组
flat boolen <任意>
 false 日期选取器被附加到元素或由事件触发
start String <任意>
 1 一天一周开始。默认:1（星期一）
mode String <任意>
 single single|multiple|range 模式
format String <任意>
 YMD 格式
position String <任意>
 bottom 位置top|left|right|bottom
onChange Callback <任意>
 日期将被改变,回调函数触发
onRender Callback <任意>
 回调函数触发时，日期的日历被渲染
onHide Callback <任意>
 回调函数时触发，日期选择器被隐藏
onShow Callback <任意>
 回调函数触发,日期选择器显示
onBeforeShow Callback <任意>
 日期选择器之前,触发回调函数
版本:
  • 3.2.1
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
require("$PATH/lib/util/calendar/datepicker");
$('.inputDate').DatePicker({
		format: 'Y-m-d',
		date: $('.inputDate').val(),
		current: $('.inputDate').val(),
		starts: 1,
		position: 'bottom',
		onBeforeShow: function() {
			$('.inputDate').DatePickerSetDate($('.inputDate').val(), true);
		},
		onChange: function(formated, dates) {
			$('.inputDate').val(formated);
		}
	});

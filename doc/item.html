item

表格渲染模板，对于大数据渲染，提高效率，有分页效果（依赖分页控件）

表格渲染模板
参数:
名称 类型 属性 默认 描述
parent String <任意>
 放置插件的容器
item Object <任意>
 表头数据 “键” 表示绑定域的ID，对应的之值表示表头名称
data Array <任意>
 数据源数据对象数组
pager boolen <任意>
 false 是否启用分页插件
per_item Number <任意>
 启用分页后每页数据的数量
display_num Number <任意>
 按钮最多显示数量
版本:
  • 2.0.1
作者:
  • susu(744276721@qq.com)
示范:
// $PATH为索引路径
var Item=require("$PATH/lib/util/item/item");
Item.init({
	parent:".table",
	item:{"ider":"ddd","ddd":"weqwe"},
	data:[{"ider":"12","ddd":"dasd"},{"ider":"1222","ddd":"da3sd"},{"ider":"1tt2","ddd":"dasd"}],
	pager:true,
	per_item:3,
	display_num:5
  })
